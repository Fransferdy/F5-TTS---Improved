# F5 TTS Gradio Interface

# This is a version of F5 TTS with a quality of life improvement

Audios generated by the state of the art open source Text to Speech generators, such as F5 TTS or GPTSoVITTS, often fail to adhere to the provided text, or sometimes produces unintelligible voice, to increase the quality of the outputs, this version of F5 TTS, applies Speech to Text to the generated audio and compares its similarity with the input text, if the similarity is below 98% it tries again, for a max of 5 tries, then it returns the generated audio that best adheres to the provided text.

This approach helps to guarantee quality of speech generation, since the resulting audio will both, better adhere to the provided text, and since the speech to text worked, it means the audio is more understandable.
